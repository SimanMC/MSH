# Minecraft Self Hosting

Minecraft Server Hosting. On your own computer!

# Requirements!

Python 3.8+;
Java 17+.


# Funcions!

Language changer;
Online/offline Display;
Start/stop system;
Local and public multyplayer;
Server IP shower and copier;
Version, Software, Gamemode, Difficulty, Players and Mods display;
Server software changer (Paper and Forge);
Custom server folder name;
Custom server message;
Custom port;
Custom version (1.8 to 1.21.11);
Custom Gamemode;
Custom difficulty;
Custom seed;
Custom max players;
Pvp, whitelist, hardcore, allow flight, online mode;
Custom Operators;
Custom RAM usage (Be careful);
Command blocks and force gamemode;
Output and command bar;
Custom mod uploader and remover.
